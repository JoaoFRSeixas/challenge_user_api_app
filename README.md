Prerequisites:

ruby "3.2.2"
"rails", "~> 7.1.3"


* Configuration
  (1) Clone this repo;
  
  (2) Install and use ruby "3.2.2"
  
  (3) Run bundle install at directory
  
  (4) Run rails db:create
  
  (5) Run "rails s"
  
  (6) Test local endpoints using "http://localhost:3000"
